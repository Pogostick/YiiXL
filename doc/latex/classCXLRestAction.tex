\subsection{Detailed Description}
\hyperlink{classCXLRestAction}{CXLRestAction} represents a REST action that is defined as a CXLRestController method. The method name is like '$<$method$>$XYZ' where $<$method$>$ is the type of HTTP request and 'XYZ' is the action name.
